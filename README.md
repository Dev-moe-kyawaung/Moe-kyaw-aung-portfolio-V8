## 🚢 Deployment — GitHub Pages

This portfolio is auto-deployed via **GitHub Actions** on every push to `main`.

### Automatic Deploy (CI/CD)

```yaml
# .github/workflows/deploy.yml
name: Deploy Portfolio to GitHub Pages
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    permissions:
      pages: write
      id-token: write
    steps:
      - uses: actions/checkout@v4
      - uses: actions/configure-pages@v4
      - uses: actions/upload-pages-artifact@v3
        with:
          path: '.'
      - uses: actions/deploy-pages@v4
```

### Manual Setup Steps

```bash
# 1. Clone the repository
git clone https://github.com/Dev-moe-kyawaung/Moe-kyaw-aung-portfolio-V8.git
cd Moe-kyaw-aung-portfolio-V8

# 2. Open locally (no build step needed)
open index.html
# or: python3 -m http.server 8080

# 3. Deploy to GitHub Pages
# → Go to repo Settings → Pages → Source: GitHub Actions
# → Push to main — auto-deploys!
```
